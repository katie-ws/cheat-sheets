Common CSS Properties

Property What it Does
background Sets all background styles simultaneously
background-color Sets element's background color
background-repeat How often background should be repeated along horizontal & vertical axes
border Sets element's border (width, style, color)
clear Determines if element must be moved below (cleared) floating elements that precede it
cursor Chooses the type of cursor displayed
filter Applies graphical effect to image e.g. blur, greyscale, color shift
font-family Specifies typeface
font-size Sets size of the font
font-weight Weight/boldness of font
letter-spacing Horizontal spacing between characters
margin Sets margin (space around an element) on all 4 sides simultaneously
overflow Determines outcome if text too large in both directions to fit pre-determined space
padding Sets padding area (space within an element) on all 4 sides simultaneously
position Specifies how element should be positioned within a document
text-align Horizontal alignment of body of text
text-transform How to capitalize text
vertical-align Vertical alignment of an element
visibility Shows or hides an element without affecting page layout
width Sets an element's width
z-index Allows elements to overlap - the higher the z-index, the more it will cover
